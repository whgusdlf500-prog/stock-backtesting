최종 업데이트: 2026-02-17

수집 항목

회원가입 기능이 없으며, 사이트는 기본적으로 이름/이메일 등 직접 식별 가능한 개인정보를 저장하지 않습니다.

브라우저 저장 정보

테스트 최근 결과와 최고 기록은 사용자 브라우저의 로컬 저장소(LocalStorage)에만 저장됩니다. 서버로 전송되지 않습니다.

사진 파일 처리

사용자가 선택한 사진은 공유 카드 렌더링을 위해 브라우저 내부에서만 읽고 처리하며 서버 저장을 하지 않습니다.

외부 서비스

  • Google AdSense: 광고 제공을 위해 쿠키가 사용될 수 있습니다.
  • 브라우저 기본 기능(Clipboard API): 사용자가 요청한 복사 기능 수행

이용자 권리

사용자는 브라우저의 사이트 데이터 삭제 기능으로 로컬 기록을 직접 삭제할 수 있으며, 페이지 내 "기록 초기화" 버튼으로도 삭제 가능합니다.
