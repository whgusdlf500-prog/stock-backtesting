사이트 소개

최종 업데이트: 2026-02-17

캐릭터 테스트 스튜디오는 오락용 성향 테스트 콘텐츠를 제공합니다. 사용자는 간단한 질문에 답하고 결과를 카드로 저장하거나 공유할 수 있습니다.

서비스 원칙

  • 결과는 재미를 위한 해석이며, 특정 개인의 가치 판단에 사용되지 않습니다.
  • 민감한 속성(건강/종교/정치/재정 상태 등)을 판단하는 문항을 제공하지 않습니다.
  • 테스트 결과는 브라우저 로컬 저장소에만 저장되며 서버 DB를 사용하지 않습니다.

사진 처리 안내

사용자가 선택한 사진은 공유 카드 이미지 생성 시 브라우저 내부에서만 사용합니다. 원본 파일과 결과 이미지를 서버로 업로드하거나 학습 데이터로 사용하지 않습니다.

면책

본 사이트의 모든 테스트는 오락 목적이며 방송/출연자/실존 인물과 직접적인 관련이 없습니다.
